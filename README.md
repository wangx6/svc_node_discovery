### API Doc
##### Get a user.
```bash
GET /user
```
```javascript
ok response
{
  ok: true,
  data: []
}

error response
{
  ok: false,
  error: []
}
```

##### Create a user
```bash
POST /user
```
```javascript
Payload
{
  name: '',
  age: '',
  gender: ''
}

ok response
{
  ok: true,
  data: []
}

{

  error: []
}
```
  
##### Update a user
```bash
PUT /user
```

```bash

```
